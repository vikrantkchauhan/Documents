=> Model :-
	-> Model is an interface that is used to store data that is rendered by a view
	-> It is a Map-Like object that stores key-value pairs where the key is the name of attribute and the value is the attribute value
	
=> ModelAndView :-
	-> ModelAndView is a class that is used to return data from a controller to a view
	-> It can store any type of data i.e. Model data or Collection data etc which can be transfered to the view
