File system/hierarchy

Windows case :\ backward slash
Linux case: / forward slash

Windows:
					C:\
	Program Files,  Users, Program x86, perflogs
			
Linux:

		        	\
	/root, /home, /boot, /etc, /usr, /bin
	
	
/home: home directory for other users
/root: home directory for root users
/boot: bootable files for linux 
/usr : default software installation directory
/etc: contains configuration files
/bin: contains commands used by all users
/sbin: contains commands used by root user
/opt: optional application software packages
/dev: essential device files like the drives etc

How to create files in linux.
cat, touch, vi, nano

1. Cat

a. Create file
b. Concatenate file
c. copy files
d. tac : displays content in reverse order

cat> file enter  //creates the file
ctrl+D to save the content and exit cat cmd

cat file // to display the content

cat >> file //to edit the existing file 

cat file1 file2 > file3 // concatenate the content of file1 and file 2 to the file3 

tac filename // displays the content in reverse order



2. touch file1 file2 etc : creates empty files 

touch // creates the empty file , change all the time

stat file // displays timestamps information of the file

timestamps: Access time(when was accessed)
			Modify time(updation time)
			change time(metadata info)
			
touch file //changes all the three timestamps

touch -a file // changes the access time only
touch -m file //changes the modify time only


3. vi editor

vi file //open the file
i // enter insert mode
esc // enter escape mode
:w // save the file
:wq //save and quit the file
:q //quit the file without saving
:q!//force quit no save
